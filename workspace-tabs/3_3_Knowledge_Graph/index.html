Product Note

页面定位

这个页面不是单纯展示样例关系，而是模拟真实使用流程：当用户输入条款、领域或业务场景后，系统先判断适合“关系链追踪”还是“跨法域对比”。关系链追踪用于解释一个条款如何关联到义务、指南、案例和业务场景；跨法域对比则围绕同一合规主题，整理不同法域下的法规、条款、义务和适用条件。技术上，跨法域对比可以通过主题子图查询完成；产品上，它呈现为更容易理解的法规对比结果。后续如接入智能体，该图谱也可以作为结构化关系底座，为问答和报告生成提供可校验的关系依据。
